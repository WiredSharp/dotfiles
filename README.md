# My dotfiles setup

use [chezmoi](https://chezmoi.io) as manager.

## Installation

- for script shell: ```sh -c "$(curl -fsLS git.io/chezmoi)" -- init --apply wiredsharp```
- for PowerShell: ```(iwr -UseBasicParsing https://git.io/chezmoi.ps1).Content | powershell -c -```
